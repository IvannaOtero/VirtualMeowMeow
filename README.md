# VirtualMeowMeow
Is a small game where you take care of tiny cat! 

What is it??!
It's basically a virtual pet, where you feed your cat, 
play with it, keep it from starving! 

How to play: 
1. Clone the repo: git clone https://github.com/IvannaOtero/VirtualMeowMeow.git
2. Open it in your editor.
3. Run it however your set up works.

Behind the scenes 
Your MeowMeow has needs like 
- Hunger
- Energy
- Happiness
